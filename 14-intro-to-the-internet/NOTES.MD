

# HTTP Request -> Sending a Letter

* Method + URL              ->      Address (where the request is getting sent)
* Headers of the Request    ->      Postage 
* Body of the Request       ->      Inside the envelope (what we want to send to the server)


* URL
    * Protocol
    * Domain
    * Path
    
* Method
    * GET
    * POST
    * PATCH
    * PUT
    * DELETE
